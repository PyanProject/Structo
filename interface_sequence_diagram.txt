title Взаимодействие пользователя с системой Structo при генерации 3D-модели

actor Пользователь
participant "Веб-интерфейс" as UI
participant "Сервер приложений" as App
participant "Сервер генерации\nмоделей" as Gen
database "База данных" as DB

Пользователь->UI: Авторизация
UI->App: Отправка учетных данных
App->DB: Проверка учетных данных
DB-->App: Подтверждение авторизации
App-->UI: Успешная авторизация
UI-->Пользователь: Отображение главной страницы

Пользователь->UI: Переход на страницу генерации
UI-->Пользователь: Отображение формы параметров

Пользователь->UI: Ввод параметров модели
Пользователь->UI: Нажатие кнопки "Генерировать"
UI->App: Отправка параметров
App->Gen: Запрос на генерацию
Gen-->App: Статус "В процессе"
App-->UI: Отображение статуса
UI-->Пользователь: Индикатор процесса генерации

Gen-->App: Результат генерации (3D-модель)
App->DB: Сохранение модели
App-->UI: Передача модели
UI-->Пользователь: Отображение готовой модели

Пользователь->UI: Взаимодействие с 3D-моделью
UI-->Пользователь: Визуальное отображение изменений

Пользователь->UI: Сохранение модели
UI->App: Запрос на сохранение
App->DB: Обновление данных модели
DB-->App: Подтверждение сохранения
App-->UI: Статус сохранения
UI-->Пользователь: Уведомление об успешном сохранении
