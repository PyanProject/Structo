ЗАКЛЮЧЕНИЕ

В данной выпускной квалификационной работе были выполнены поставленные задачи по разработке распределенной системы генерации 3D-моделей на основе текстовых описаний (промптов) с использованием модуля Shape-E.

В ходе работы был проведен анализ предметной области генерации трехмерных моделей с помощью искусственного интеллекта, сформированы необходимые функциональные требования к разрабатываемой информационной системе. Проведен подробный анализ существующих решений и технологий в данной области, выбраны оптимальные инструменты для реализации поставленных задач.

Разработанная система включает в себя следующие компоненты:
1. API-сервер на локальном ПК, выполняющий генерацию 3D-моделей с использованием GPU (shape_server.py)
2. Веб-приложение на VDS-сервере, предоставляющее пользовательский интерфейс для взаимодействия с системой (app.py)
3. Модуль удаленного взаимодействия между компонентами системы (remote_generator.py)
4. База данных для хранения информации о пользователях и созданных моделях

В системе реализованы следующие ключевые функции:
- Регистрация и аутентификация пользователей
- Создание 3D-моделей на основе текстовых описаний
- Визуализация и скачивание сгенерированных моделей
- Отслеживание процесса генерации в режиме реального времени
- Управление пользовательским профилем и историей созданных моделей
- Система подписок для различных категорий пользователей

Для создания системы использовались современные технологии:
- Python в качестве основного языка программирования
- Flask для создания веб-приложения и API
- PyTorch и Shape-E для генерации 3D-моделей
- SQLAlchemy для работы с базой данных
- HTML, CSS и JavaScript для разработки пользовательского интерфейса
- Nginx для настройки прокси-сервера и обеспечения безопасности

В результате выполнения работы была создана полноценная система, позволяющая пользователям без специальных навыков в 3D-моделировании создавать трехмерные модели на основе текстовых описаний. Система успешно решает проблему доступности технологий генеративного искусственного интеллекта для широкого круга пользователей, предоставляя интуитивно понятный интерфейс и эффективную инфраструктуру для обработки запросов на генерацию.

Проведенное тестирование системы показало её стабильную работу, эффективность генерации моделей и корректность взаимодействия между компонентами. Разработанная архитектура обеспечивает масштабируемость системы и возможность её дальнейшего развития.

Выполненная работа имеет практическую значимость и может быть использована в образовательных, творческих и коммерческих целях, предоставляя пользователям новые возможности в области 3D-моделирования с использованием технологий искусственного интеллекта.
