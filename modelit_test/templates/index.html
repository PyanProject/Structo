Как это работает?

ModelIT использует многоэтапный процесс для генерации 3D моделей:

  1. Кодирование текста - Текстовый запрос преобразуется в эмбеддинги, которые содержат семантическую информацию о запрашиваемом объекте.
  2. Диффузионная модель - Генерирует латентное представление 3D объекта на основе текстового эмбеддинга.
  3. 3D декодер - Преобразует латентное представление в вокселизированную 3D сетку.
  4. Рендеринг - Создает визуализацию полученной 3D модели для предварительного просмотра.

Возможности системы

  • Генерация 3D моделей из текстовых описаний
  • Поддержка различных параметров генерации (guidance scale, seed)
  • Сохранение моделей в различных форматах (.obj, .glb)
  • Возможность предварительного просмотра сгенерированных моделей
  • Обучение модели на собственных данных
