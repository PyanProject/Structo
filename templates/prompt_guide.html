Как писать промпты для генерации 3D-моделей

Этот гайд поможет вам создавать эффективные запросы для генерации 3D-моделей в Structo. Следуйте этим рекомендациям для получения наилучших результатов.

Основные параметры

  • Цвет: Укажите цвет объекта.
  • Форма: Опишите форму объекта.
  • Размер: Укажите размер объекта.
